In this game you control a spaceship in a field of asteroids. If you get hit by an 
asteroid the game ends. You can shoot the asteroids but if you blow one up, two more
spawn in the vicinity.

1. Rocket Science
- up and down arrow keys give thrust
- left and right arrow keys apply torque
2. Rubberoids
- asteroids can collide with each other 
3. Boom
- ship explodes when hit by asteroid
- asteroids explode when shot
4. Flames
- thrusters have exhaust flames
5. Shoot 'em up
- Space bar shoots a bolt in the direction the ship is facing with set velocity
