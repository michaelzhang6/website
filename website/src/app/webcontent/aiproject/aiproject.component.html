Marth Bot

For my Artifical Intelligence Practicum Project Course, my teammates and I created a reinforcement learning AI that learns how to play Super Smash Brothers Melee. We used the Dolphin GameCube emulator to run the game and get game state information. The AI controlled a Marth character in the game by inputting commands into a virtual controller. We implemented a single deep Q neural network (DQN) for the AI to be able to learn from its in game experiences. The AI was able to attack a target, dodge enemy attacks, as well as recover from being knocked off the stage.
